## 🍽️ About The Project

**Full Stop Cafe** is a fully responsive, modern restaurant website for a **24×7 cafe in Alpha-I, Greater Noida**.  
The website features a **3D animated background**, interactive food menu, real-time item details modal, online ordering links (Zomato/Swiggy), food gallery, combo deals, and complete business information.

Built with pure **HTML, CSS, and JavaScript** — no frameworks, fast loading, and easy to customize.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🎨 **Premium UI/UX** | Dark theme with fire-orange accents, custom cursor, smooth animations |
| 🧊 **3D Background** | Rotating torus knot + floating particles for immersive experience |
| 📱 **Fully Responsive** | Works on desktop, tablet, and all mobile devices |
| 🍕 **Dynamic Menu** | 60+ food items with categories (Rolls, Noodles, Rice, Momos, Tandoor, Indian Mains, etc.) |
| 🔍 **Category Filter** | One-click filtering — no page reload |
| 🖼️ **Food Gallery** | Aesthetic gallery showcasing signature dishes |
| 📦 **Item Modal** | Click any menu card to see full details with veg/non-veg badge |
| 🛵 **Order Integration** | Direct links to Zomato, Swiggy, and phone order |
| 📍 **Location & Contact** | Google Maps link, 24×7 timing, phone, social handles |
| ⚡ **Performance** | 100% HTML/CSS/JS — no dependencies, instant load |
