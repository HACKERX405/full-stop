## 🍽️ About The Project

**Full Stop Cafe** is a fully responsive, modern restaurant website built for a 24×7 cafe in **Greater Noida**.  
The website features a **3D animated background**, interactive food menu with categories, real-time item details modal, online ordering integration (Zomato/Swiggy), and a complete business showcase.

This project is perfect for small restaurants, cafes, or cloud kitchens looking to establish a premium online presence without heavy backend complexity.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🎨 **Modern UI/UX** | Dark-themed design with fire-orange accents, custom cursor, and smooth animations |
| 🧊 **3D Background** | Rotating torus knot + particle effects for a futuristic feel |
| 📱 **Fully Responsive** | Works flawlessly on desktop, tablet, and mobile devices |
| 🍕 **Dynamic Menu** | 60+ food items categorized (Rolls, Noodles, Rice, Momos, Tandoor, Indian Mains, etc.) |
| 🔍 **Category Filter** | One-click filtering by food type — no page reload |
| 🖼️ **Food Gallery** | Aesthetic gallery section showcasing signature dishes |
| 📦 **Item Modal** | Click any menu card to see full details (price, description, veg/non-veg badge) |
| 🛵 **Order Integration** | Links to Zomato, Swiggy, and direct phone order |
| 📍 **Location & Contact** | Google Maps link, 24×7 timing, phone, and social media handles |
| ⚡ **Performance** | Pure HTML/CSS/JS — no frameworks, fast loading |
